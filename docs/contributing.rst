Contributing
------------

If you have found a bug or an issue in the software then please raise an issue on the
repository's `Issues <https://github.com/amckenna41/protpy/issues/>`_ tab.

If you would like to contribute any functional or feature changes to the software, please make a pull
request on the `repository <https://github.com/amckenna41/protpy/>`_.

Any other queries or issues, please contact me via email: amckenna41@qub.ac.uk 😁
